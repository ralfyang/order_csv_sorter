# 주문리스트 중복 정렬 및 SKU 분리를 위한 개발 by ChatGPT
## golang 개발 및 sorting script 제작
* `ChatGPT`를 활용한 개발
* CSV 파일중 `[1+1]` 또는 `[2+2]` 항목의 order가 있을 경우, 각자의 SKU로 나누어서 행을 정렬하는 도구
* csv 파일 업로드를 통해 처리 완료된 파일을 바로 download 하여 사용
